<h2 align="center">
  Formula
</h2>
  
<p align="center">
  Rewards balance to claim rewards at:
</p>

<br>

$$
Balance_\mathrm{rewards} \ = 
  \ \cfrac
    {3 \ \times \ \left(
      Fee \ + 
        \ \sqrt{
          Fee \ \times \ ( (4 \ \times \ Balance_\mathrm{staked}) \ + \ Fee ) 
        } 
    \right) }
    {4}
$$
