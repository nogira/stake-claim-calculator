<p align="center">Rewards balance to claim rewards at:</p>

$$
Balance_\mathrm{rewards} \ = 
  \ \cfrac
    {3 \ \times \ \left(
      Fee \ + 
        \ \sqrt{
          Fee \ \times \ ( (4 \ \times \ Balance_\mathrm{staked}) \ + \ Fee ) 
        } 
    \right) }
    {4}
$$
